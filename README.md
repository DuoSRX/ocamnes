# OCamNes

Yes. Yet another NES Emulator.

Currently passes the Nestest suite up to unofficial opcodes.

## Controls

* Up: up arrow
* Down: down arrow
* Left: left arrow
* Right: right arrow
* A: Z
* B: X
* Start: enter
* Select: left shift

## Games running mostly glitch-less:

* Super Mario Bros
* Donkey kong
* Balloon fight
* Lode runner
